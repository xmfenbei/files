# 判断

```
@NotNull(message = "名称不能为空")
private String name;
```

```
@Size(min = 1, max = 200, message = "名称长度为1~200")
private String name;
```
长度的判断必须建立在name有value的情况下才有效，不能用min=1来判断name必填。

```
@Size(min = 1, max = 200, message = "名称长度为1~200")
@NotNull(message = "名称不能为空")
private String name;
```
